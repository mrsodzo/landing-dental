# Лендинг стоматологической клиники «Дент-Люкс»

Одностраничный лендинг для демонстрации услуг клиники, врачей и сбора заявок через форму записи. Форма отправляет данные в Telegram-бота.

## Что использовано

- **HTML5 / CSS3 / JS** — чистый стек, без сборщиков и фреймворков
- **CSS-переменные**, Grid, Flexbox, медиазапросы — адаптив под мобильные/планшеты/десктоп
- **Font Awesome 6** — иконки
- **Unsplash** — демо-фото врачей и клиники
- **Яндекс.Карты** — встроенная карта с меткой
- **Telegram Bot API** — получение заявок с формы в чат

## Запуск

Откройте `index.html` в браузере. Для продакшна достаточно hosted-статики (GitHub Pages, Vercel и т.д.).

## Настройка Telegram-уведомлений

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите `BOT_TOKEN`
2. Узнайте `CHAT_ID`, куда слать заявки (личный чат или группа)
3. Отредактируйте `js/config.js`:
   - `TELEGRAM_BOT_TOKEN` — токен бота
   - `TELEGRAM_CHAT_ID` — ID чата
4. Файл `js/config.js` уже добавлен в `.gitignore`, чтобы не коммитить секреты.

```js
window.TELEGRAM_BOT_TOKEN = '123456:ABC-DEF...';
window.TELEGRAM_CHAT_ID = '-1001234567890';
```

## Особенности

- Фиксированная шапка с бургер-меню на мобильных
- Плавный скролл к секциям с учётом высоты шапки
- Маска ввода телефона `+7 (___) ___-__-__`
- Валидация обязательных полей формы
- Фолбэк на телефон, если Telegram не настроен
